# Cruisin Craftsmen Calculator

## Project Description
This project is a calculator designed for craftsmen to assist with various calculations required in their work. Whether it's measurements, material estimates, or time calculations, this calculator provides a user-friendly interface to perform these tasks efficiently.

## How to Use the Calculator
1. **Input Values:** Enter the values required for your calculation in the provided fields.
2. **Select Operation:** Choose the operation you want to perform (e.g., addition, subtraction, etc.).
3. **Calculate:** Click the calculate button to get the result displayed on the screen.
4. **Reset:** Clear the input fields to start a new calculation.

## Code Structure
- **src/**: This directory contains the main source code of the calculator. It includes all the necessary functionalities to perform calculations.
- **components/**: Here you'll find the different components used in the calculator’s UI. Each component handles a specific part of the user interface.
- **styles/**: This folder contains all the CSS files for styling the calculator, ensuring a coherent visual experience.
- **index.html**: This is the main HTML file that loads the application.

Feel free to explore the code and understand how the different parts interact to create the calculator functionality!
